## 接口文档

### 1.管理员获取语言列表

#### 1.1 功能描述

管理员获取语言列表

#### 1.2 请求说明

> 请求方式：GET
> 请求URL ：/language/admin/languages_manage/{pageNum}
>
> URL参数含义：pageNum表示当前页号

#### 1.3 请求参数

无

#### 1.4 返回结果接收

```json  
{
  "code": 200,
  "data": {
    "adminId":"15180600314",
    "adminName":"辛某人"
  }
}
```

#### 1.5 返回参数

| 字段      | 字段类型 | 字段说明   |
| --------- | -------- | ---------- |
| adminId   | string   | 管理员ID   |
| adminName | string   | 管理员密码 |

#### 1.6 错误状态码

| 状态码 | 说明                     |
| ------ | ------------------------ |
| 200    | 操作成功                 |
| 2222   | 信息验证失败，请重新登录 |
